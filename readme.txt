Plugin that allow you to hide update notifications for each plugin installed

== Description ==

Plugin that allow you to hide update notifications for each plugin installed. Choose which plugins you want to hide from a simple list and Wordpress will not show you further notifications for those plugins.

== Installation ==

1. Unzip and upload the plugin in your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Drag & Drop the widget in your sidebar and customize your options
